# MathMaster 
## MathMaster - это проект-приложение, в виде игры, который помогает выучить или вспомнить таблицу умножения. Многие дети и даже взрослые сталкиваются с трудностью при заучивании таблицы умножения. Обычная зубрёжка — скучная и неэффективная. Проект решает эту проблему, превращая обучение в интерактивную игру.

##  Целевая аудитория:
Проект ориентирован на несколько групп пользователей:

• Младшие школьники (2–4 классы) — основная аудитория, которая как раз учит таблицу умножения.

• Взрослые — для поддержания мозга в тонусе (нейротренажёр).

##  Цели и задачи проекта:
### Цель:
Создать простой, понятный и полезный тренажёр для закрепления знания таблицы умножения в игровой форме.

### Задачи:
• Разработать удобный графический интерфейс (чтобы ребёнку было интуитивно понятно).

• Реализовать генерацию случайных примеров (чтобы каждый раз было по-новому).

• Добавить систему проверки ответов с мгновенной обратной связью.

• Внедрить статистику (счётчик правильных ответов и попыток).

• Сделать сохранение истории в файл.

• Добавить возможность ввода никнейма (персонализация).

• Обеспечить защиту от некорректного ввода (буквы, пустые поля).

##  Функционал проекта:
### 1. Приветствие и персонализация
• При запуске появляется окно с полем для ввода никнейма.

• Если ничего не ввести, по умолчанию присваивается имя "Пользователь".

• Никнейм отображается в интерфейсе и сохраняется в логах.

### 2. Игровой процесс
• Генерация двух случайных чисел от 2 до 9.

• Отображение примера на экране крупным шрифтом: 5 * 7 = ?.

• Поле для ввода ответа.

• Возможность отправить ответ:

• Кнопкой "Проверить"

• Клавишей Enter

• После ответа появляется сообщение:

 Верно! (зелёный цвет)

 Ошибка! Правильный ответ: ... (красный цвет)

### 3. Управление игрой
• Кнопка "Новый вопрос" — пропустить текущий пример и получить новый.

• Кнопка "Выйти" — завершение игры с показом итогов.

• Автоматическая смена вопроса через 1 секунду после ответа.

### 4. Статистика в реальном времени
• Отображается в нижней части окна:

• Количество правильных ответов

• Общее количество попыток

• Процент успеха

### 5. Сохранение истории
• Все действия пользователя записываются в файл log.txt.

### 6. Защита от ошибок ввода
• Проверка на пустой ввод.

• Проверка на ввод букв и символов (только цифры).

• Сообщение об ошибке с подсказкой.
