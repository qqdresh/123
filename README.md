#  🚀MathMaster 
## MathMaster - это проект-приложение, который помогает выучить или вспомнить таблицу умножения.

## 🧒Целевая аудитория:
Проект ориентирован на:

Младших школьников (2–4 класса) - основная аудитория, которая как раз учит таблицу умножения.

## 🎯Цели и задачи проекта:
### Цель:
Создать простой, понятный и полезный тренажёр для закрепления знания таблицы умножения в игровой форме.

### Задачи:
• Сделать Use Case диаграмму

• Блок схему

• Реализовать программу

• Оформить текст

• Оформить презентацию

## ⚙️Функционал проекта:
• Программа должна давать задачи на умножение/деление чисел ( пользователь выбирает сам )

• Проверять ответ
