# MathMaster
Какую проблему решает проект: Помогает детям  быстро выучить таблицу умножения в игровой форме, без скучной зубрежки. Развивает навыки устного счета.
Целевая аудитория: школьники
Цели и задачи:

Цель: Создать простой интерактивный тренажер для закрепления знания таблицы умножения.

Задачи:

1. Генерировать случайные примеры.
2. Проверять правильность ответа пользователя.
3. Вести подсчет правильных и неправильных ответов.
4. Реализовать несколько уровней сложности.
Функционал:

Выбор уровня сложности (например: только умножение, умножение и деление).

Генерация случайных чисел (от 2 до 9).

Показ примера на экране.

Ввод ответа пользователем с клавиатуры.

Проверка ответа и вывод результата (Верно/Неверно).

Подсчёт очков (счетчик правильных ответов).

Возможность завершить игру и увидеть итоговый результат.
