# MathMaster
# Какую проблему решает проект:
Помогает детям  быстро выучить таблицу умножения в игровой форме, без скучной зубрежки. Развивает навыки устного счета.

# Целевая аудитория:
Школьники 1-4 класса обучения

# Цели и задачи:

Цель: Создать простой интерактивный тренажер для закрепления знания таблицы умножения.

Задачи:

1. Генерировать случайные примеры.
2. Проверять правильность ответа пользователя.
3. Вести подсчет правильных и неправильных ответов.
4. Реализовать несколько уровней сложности.
5. Помочь школьникам в обучении
   

# Функционал:

Выбор уровня сложности (например: только умножение, умножение и деление).
1. Генерация случайных чисел (от 2 до 9).
2. Показ примера на экране.
3. Ввод ответа пользователем с клавиатуры.
4. Проверка ответа и вывод результата (Верно/Неверно).
5. Подсчёт очков (счетчик правильных ответов).
6. Возможность завершить игру и увидеть итоговый результат.
