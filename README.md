# MathMaster 
## MathMaster - это проект-приложение, в виде игры, который помогает выучить или вспомнить таблицу умножения. Обычная зубрёжка — скучная и неэффективная. Проект решает эту проблему, превращая обучение в интерактивную игру.

##  Целевая аудитория:
Проект ориентирован на несколько групп пользователей:

• Младшие школьники (2–4 классы) — основная аудитория, которая как раз учит таблицу умножения.

• Взрослые — для поддержания мозга в тонусе (нейротренажёр).

##  Цели и задачи проекта:
### Цель:
Создать простой, понятный и полезный тренажёр для закрепления знания таблицы умножения в игровой форме.

### Задачи:
• Сделать Use Case диаграмму

• Блок схему

• Реализовать программу

• Оформить текст

• Оформить презентацию

##  Функционал проекта:
• Программа должна давать задачи на умножение чисел
• Проверять ответ
